# THE LEAKED GUI TRUE!!!
A mod based on the OreUI leak on [Reddit](https://www.reddit.com/r/Minecraft/comments/1hpy4ao/new_inventory_ui_overhaul_leaked/#lightbox), using files found in the Minecraft Preview OreUI assets.

Currently, the additions are:
 * Creative Mode Inventory
 * Inventory
 * Furnace/Blast Furnace/Smoker
 * Shulker Boxes
 * Chests:
   * Ender chest
   * Normal chest
   * Copper chests
   * Barrel
